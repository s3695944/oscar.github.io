Learning National Culture Mobile App

Overview and Motivation

I will create an application to learn and language and culture The functions of this app are: uploading video, user registration, online communication, introduction of culture, online language class Users interact online, share learning experiences, or upload learning materials to exchange learning information with other users.

I want to make a mobile app that teaches the language of each country and shows the culture of each country. There are 799,371 international students were enrolled in education programs in Australia. Of these, there were 350,472 international students enrolled in the higher education sector. International students can usually communicate in English But when you exchange national cultures They cannot accurately describe in English This app can support national cultures in multiple languages. And there are pictures, videos, and music to introduce the culture of each country. Students can also use this app to learn foreign languages.

my APP

Open this mobile app and first see the LOGO.
Then enter the HOME interface. The home interface has 4 buttons.
The first one is the login button. Click to enter the login interface.
I want to record user preferences by creating an account for the user.
Login can record the webpage that the user last searched.                                                                                                                                                                   
The second button is a national culture introduction.
This interface has a search function. The search function can identify the languages ​​of multiple countries.
And this page has videos and music from national culture.
These files are stored in the database. Users can add videos and music from the web page after logging in.                                                                                                                                                                   
The third interface is the language teaching interface.
I will download free and some paid teaching documents from the Internet.
Users can also add teaching files
This interface also has a search function. These files are stored in the database
I can classify the course into different levels. user can choose the course difficulty according to the learning situation. The user can receive the best teaching.
These files are stored in different databases depending on the level and language.                                                                                                                                                                                              
The fourth interface is the user interface.
Users can chat on this interface after logging in.
This interface has various language areas. Users can chat according to their own preferences
And you can find translations in the teaching area.
The chat content is stored in the database.                                                                                                                                                                   
At the same time, this app also has an "error" button.
Users can leave a message on this interface
